# Feel36 - Приложение 36 вопросов для сближения

Мини-приложение для Telegram, созданное для укрепления отношений с помощью набора из 36 специальных вопросов.

## О проекте

Feel36 - это интерактивное приложение, разработанное на базе научных исследований, которые показывают, что 36 специальных вопросов способны создать глубокую эмоциональную связь между людьми. Вопросы разделены на три блока, с каждым блоком они становятся все более глубокими и откровенными.

## Технологии

- React
- TypeScript
- Vite
- Tailwind CSS
- Vercel (хостинг)

## Особенности приложения

- Пошаговое введение перед началом
- 36 вопросов, разделенных на 3 блока
- Упражнение на зрительный контакт после каждого блока вопросов
- Современный и отзывчивый интерфейс

## Деплой

Приложение развернуто на Vercel и доступно по адресу: [Feel36](https://feel36.vercel.app/)
